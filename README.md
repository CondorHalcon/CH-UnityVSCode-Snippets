# unity-vscode-snippets README

Unity VSCode Snippets is a Unity C# snippet extention built for VSCode.

Author: [@CondorHalcon](https://github.com/CondorHalcon)

### Installation
1. Download a [released](https://github.com/CondorHalcon/unity-vscode-snippets/releases) version
2. Extract it
3. Move it to:
    - __Windows__ `%USERPROFILE%/.vscode/extentions`
    - __macOS__ `~/.vscode/extentions`
    - __Linux__ `~/.vsode/extentions`
4. Restart VSCode

__Note__: The repository my have unknown bugs that would be difficult to track down as the code may have been edited. Downloading the repository is highly discouraged.

## Features

- Unity C# snippets

## Requirements

### Minimum
